Ajax Curency

Ajax Currency - це мультимодульний веб-додаток, який складається з двох частин, об'єднаних однією базою даних MySQL. Перша частина буде написана на Java Spring, а інша - на PHP. Різниця між ними полягатиме в тому, що додаток на Spring буде базуватися на звичайній парадигмі запит/відповідь (request/response), тоді як сервер на PHP буде використовувати технологію асинхронної передачі інформації AJAX для отримання та додавання інформації в базу даних.

Основною функцією проекту є отримання інформації щодо курсу валют та її запис до бази даних. У Java Spring-додатку буде використовуватися для цього дані по долару, тоді як в PHP-реалізації - для євро. Головною метою проекту є дослідження технічних відмінностей між двома основними технологічними підходами: "запит/відповідь" (request/response) в Java Spring і асинхронною передачею даних за допомогою AJAX в PHP.

P.S.

Проект знахидиться на стадії research, ніякої готової функціональності ще немає.
